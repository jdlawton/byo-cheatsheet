CSS Border Properties

Style CSS Property Example
Dotted border-style: dotted; A dotted border
Dashed border-style: dashed; A dashed border
Solid border-style: solid; A solid border
Double border-style: double; A double border
Groove border-style: groove; A groove border
Ridge border-style: ridge; A ridge border
Inset border-style: inset; A inset border
Outset border-style: outset; A outset border
None border-style: none; A none border

Border Width

Border width can be set as a specific size (in px, pt, cm, em, etc) or by using one of the three pre-defined values: thin, medium, or thick.

CSS Code Example
border-style: solid;
border-width: 5px;		 5px border-width
border-style: solid;
border-width: medium;		 medium border-width
border-style: dotted;
border-width: 2px;		 2px border-width
border-style: dotted;
border-width: thick;		 thick border-width

Border Color

The color can be set by:

  • name - specify a color name, like "red"
  • HEX - specify a HEX value, like "#ff0000"
  • RGB - specify a RGB value, like "rgb(255,0,0)"
  • HSL - specify a HSL value, like "hsl(0, 100%, 50%)"
  • transparent
CSS Code Example
border-style: solid;
border-color: red;		 Red border
border-style: solid;
border-color: green;		 Green border
border-style: dotted;
border-color: blue;		 Blue border

Border Sides

There are also properties for specifying each of the borders (top, right, bottom, and left)

Rounded Borders

The border-radius property is used to add rounded borders to an element

CSS Code Example
border-style: solid; Normal border
border-style: solid;
border-radius: 5px;		 Rounded Border
border-style: solid;
border-radius: 10px;		 Rounder Border
border-style: solid;
border-radius: 15px;		 Roundest Border

Border Shorthand

The border property is a shorthand property for the following individual border properties:

  • border-width
  • border-style (required)
  • border-color
CSS Code Example
border: 6px solid red Some text
border-left: 6px solid red Some text
border-bottom: 6px solid red Some text
